MODULE PRESTASHOP MTARGET SMS 

Le module est compatible avec toutes les version 1.6 et 1.7 de Prestashop

Pour l'installer, assurez vous de disposer d'une des versions listées ci-dessus. (Pour connaitre votre version # actuelle, rendez-vous dans l'onglet CONFIGURER > Paramètres avancés > Informations)

Pour l'installer,
Uploadez le fichier via l'onglet Modules > Installer un Module et glissez ou selectionnez votre fichier .zip
Une fois le fichier envoyé, cliquez sur installer et accédez à la page d'accueil du module

Vous n'avez plus qu'a vous inscrire pour commencer à envoyer vos premiers SMS !

Le module etant installé nativement sur les version 1.7.x de Prestashop, il vous suffira simplement de l'activer via # l'onglet Modules

/!\ IMPORTANT /!\ 

Pour un fonctionenemnt optimal du module, il est conseillé d'activer la saisie obligatoire du numéro de téléphone pour vos clients
Pour cela, accédez à l'onglet VENDRE > Clients > Adresses. En bas de page, cliquez sur "Définir les champs requis pour cette section", puis cochez "phone_mobile"
